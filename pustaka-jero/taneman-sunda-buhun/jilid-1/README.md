# Pustaka Taneman Sunda Buhun - Jilid 1

Repo ieu pikeun nyusun pangeling taneman, pohon, bibit, wanci tanam, lemah cai, jeung sumber verifikasi.

## Rambu

- Ulah ngaku jadwal tanam Kalender Sunda lamun konversina teu acan diverifikasi.
- Ulah nyokot taneman langka ti alam.
- Cocokna pohon jeung jelema dibaca tina laku, lahan, cai, jeung kamampuh miara; lain ramalan.
- Sumber dibagi: A naskah/arsip, B regulasi/jurnal/buku, C adat hirup, D tafsir aman, X teu didukung.

## File utama

- `md/tabel-taneman.md` - database awal tina tab Tabel_Taneman.
- `md/pangeling.md` - pangeling jeung rambu workbook.
- `md/sumber.md` - rujukan jeung URL.
- `md/riksa-wanci.md` - peta usum tanam.
- `md/jalur-bibit.md` - cara meunangkeun bibit legal.
- `csv/` - versi tabel ringan supaya GitHub bisa muka langsung tanpa download Excel.

## Lokasi arsip

Paket ieu disimpen di:

`pustaka-jero/taneman-sunda-buhun/jilid-1/`

Alesan: ieu database awal Pustaka Hirup pikeun taneman, pohon, bibit, wanci tanam, lemah cai, jeung sumber verifikasi. Ieu bagian tina Pustaka Jero sabab maca hirup tina lemah cai, raga taneman, jalur bibit legal, jeung wanci lokal.

## Catetan penting

- Ieu Jilid 1 / database awal, lain daptar final sakabeh taneman Sunda.
- Tanggal Kalender Sunda teu meunang dieusi angka palsu lamun konversina tacan diverifikasi.
- Taneman langka/dilindungi teu meunang dicokot ti alam.
- Tingkat sumber kudu dipisahkeun: A/B/C/D/E/F/X.
