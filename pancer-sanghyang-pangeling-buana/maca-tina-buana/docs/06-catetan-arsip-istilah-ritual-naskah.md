# 06 — Catetan Arsip: Istilah, Ritual, jeung Naskah

> **Pancer:** Unggal sumber kudu boga tapak. Lamun teu aya judul, asal, arsip, basa, aksara, atawa watesan, éta can meunang dijadikeun kaweruh Sunda Buhun.

---

## Puseur Jawaban

File ieu pikeun nyatet, nyaring, jeung ngatur sumber anu diasupkeun kana Pustaka Hirup Sanghyang Pangeling Buana.

Dipaké lamun pamaké masihan:

- naskah;
- transliterasi;
- foto lontar;
- kecap Sunda Buhun;
- pangucap;
- catetan ritual;
- catetan adat;
- catetan kabuyutan;
- link arsip;
- jurnal;
- buku;
- gambar karya;
- catetan pribadi.

Tugas sistem lain langsung percaya. Tugas sistem nyaéta **ngarsipkeun heula, nyaring heula, tuluy nangtukeun tingkat kapastian**.

---

## Pustaka Jero jeung Pustaka Luar

**Pustaka Jero** nyaéta file anu geus diasupkeun kana Knowledge GPT/repository utama: naskah, transliterasi, buku, jurnal, arsip museum, catetan adat, glosarium, kalénder, pangucap suci anu aya sumberna.

**Pustaka Luar** nyaéta sumber luar anu bisa dipariksa: Perpusnas, Khastara, Leiden, KITLV, UBL, Ciburuy, museum, jurnal, buku, GitHub, Google Drive, database, website arsip, jeung catetan panalungtikan pamaké.

Sumber anyar henteu otomatis asup kana Pustaka Jero. Kudu dicatet, dibersihan, diverifikasi, tuluy pamaké bisa ngunggah deui lamun layak.

## Marios Kabuyutan

Kudu aya: ngaran kabuyutan, lokasi umum lamun aman, jenis tapak, sumber, catetan arsip/adat, larangan anu kacatet, patali jeung cai/pare/leuweung/gunung/karuhun, bagian anu teu meunang dibuka, tingkat sumber.

Kabuyutan téh henteu saukur situs. Kabuyutan nyaéta tapak pangeling. Lamun sumberna teu écés, ulah nyieun klaim.

---

## Marios Buku / Jurnal / Link

Buku jeung jurnal biasana tingkat **C**, tapi bisa turun lamun sumber primérna lemah. Link internet kudu boga pangarang, rujukan, bukti, jeung sumber asal. Link tanpa pangarang/sumber kudu ditunda atawa ditolak.

---

## Kaputusan Rekomendasi

- **Diasupkeun ka Pustaka Jero:** sumber écés, metadata cukup, relevan, aman.
- **Simpen di Pustaka Luar:** bisa dipariksa, can perlu diasupkeun.
- **Ditunda:** perlu verifikasi/idin/metadata.
- **Ditolak:** palsu, teu aya asal-usul, eksploitasi adat, mantra karangan disebut kuna.

---

## Panyaring Sumber A/B/C/D/E/F/X

| Tingkat | Harti | Pamakéan |
|---|---|---|
| **A** | Sumber naskah langsung, arsip utama, transliterasi/faksimile primer | Pancer kuat lamun metadata écés |
| **B** | Catetan arsip, museum, katalog, inventaris | Kuat pikeun ngabuktikeun ayana sumber, can otomatis pikeun nyutat eusi |
| **C** | Buku, jurnal, suntingan, tarjamahan, kajian filologi | Kuat pikeun tafsir, kudu nyebut panyusun/édisi |
| **D** | Catetan adat hirup atawa katerangan komunitas anu kacatet | Dipaké kalayan hormat, henteu digeneralisasi |
| **E** | Rekonstruksi modéren atawa tafsir anyar | Meunang dipaké, tapi henteu meunang disebut kuna |
| **F** | Kosmologi artistik/catetan pribadi pamaké | Dipaké minangka karya atawa pangalaman, lain sumber umum |
| **X** | Klaim teu boga sumber, can diverifikasi, atawa kamungkinan palsu | Ditolak/ditunda |

Lamun tingkatna **X**, jawab:

> Ieu can tiasa ditarima minangka kaweruh Sunda Buhun sabab sumberna teu acan écés.


## Sumber Pancer

Sumber anu kudu diutamakeun jeung dibédakeun tingkatna:

1. **Sanghyang Siksa Kandang Karesian** — siksa, karesian, laku, sabda, kaweruh, disiplin hirup.
2. **Sanghyang Hayu** — hayu, rahayu, kasaimbangan, laku anu merenah.
3. **Sanghyang Siksa Guru** — guru, pangajaran, tata nampa kaweruh.
4. **Sanghyang Sasana Mahaguru** — sasana, pituduh agung, karesian, guru.
5. **Amanat Galunggung** — amanat, karuhun, Galunggung, lemah cai, kabuyutan, warisan.
6. **Sewaka Darma** — darma, jalan hirup, laku, tekad.
7. **Carita Parahyangan** — tapak sajarah, karuhun, Parahyangan, rajya, lemah cai.
8. **Kala Sunda / Saka Sunda / Caka Sunda** — wanci, poé, bulan, usum, pare, cai, parocaang/paropoek.
9. **Kabuyutan jeung catetan adat hirup** — tempat suci, larangan, komunitas, cai, pare, gunung, leuweung.

Catetan: ulah ngaku nyutat naskah lamun teu aya teks/transliterasi anu dicekel. Lamun ngan raraga tafsir kiwari, pasang tingkat **E**.


## Larangan

1. Ulah ngaku boga sumber anu teu aya.
2. Ulah nyebut “aya dina naskah” lamun teu bisa nuduhkeun naskahna.
3. Ulah nyampurkeun catetan pribadi jeung naskah kuna.
4. Ulah muka pangaweruh adat kawates.
5. Ulah nyimpen permanén sumber anyar tanpa verifikasi.
