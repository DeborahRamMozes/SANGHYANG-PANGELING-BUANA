# 05. Pangucap Sunda Buhun Tina Naskah

## Fungsi Dokumen

Dokumen ieu pikeun ngumpulkeun, misahkeun, jeung mariksa pangucap Sunda Buhun anu asalna tina naskah atawa arsip. Ieu lain tempat nyieun ungkara palsu anu dibungkus siga kuna, sabab éta téh lain kabudayaan, éta kosmetik palsu pikeun kamiskinan sumber.

## Kategori Pangucap

| Kategori | Harti |
|---|---|
| Kutipan naskah | Pangucap langsung tina naskah/arsip. |
| Alih aksara | Robahan tina aksara aslina kana aksara Latin. |
| Alih basa | Tarjamahan kana basa Sunda/Indonesia kiwari. |
| Tafsir | Bacaan makna, kudu dipisah tina kutipan. |
| Rekonstruksi | Rarangkén deui tina sababaraha sumber. |
| Ciptaan anyar | Ungkara anyar anu diilhami ku Sunda Buhun, tapi teu ngaku jadi kuna. |

## Format Entri

```text
Judul/pangucap:
Teks asli lamun aya:
Alih aksara:
Alih basa:
Sumber naskah/arsip:
Kaca/baris lamun aya:
Konteks:
Tingkat sumber:
Tafsir:
Watesan:
```

## Aturan Ketat

1. Pangucap tina naskah kudu boga sumber.
2. Tafsir teu meunang dicampur jeung kutipan.
3. Ciptaan anyar kudu disebut ciptaan anyar.
4. Pangucap anu teu aya sumberna kudu asup kana E, F, atawa X.
5. Ulah nyieun “kuna-kunaan” pikeun gaya.

## Catetan

Sunda Buhun teu perlu dipalsukeun. Nu perlu téh dibaca deui kalawan ati, sumber, jeung kahormatan.
