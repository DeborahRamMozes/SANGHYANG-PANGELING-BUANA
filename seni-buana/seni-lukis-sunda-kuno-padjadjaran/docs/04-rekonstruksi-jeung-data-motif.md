# Rekonstruksi jeung Data Motif

## Aman Disebut

- Aya data ngeunaan rupa-rupa tulis jeung ahli rupa dina SSKK.
- Aya istilah balé tulis dina Carita Ratu Pakuan.
- Aya istilah tinulis, pinarada, warna, jeung cacaritaan dina Carita Parahyangan.
- Falak Budha / Raffles Java 1 miboga unsur gambar rasi, tabel, jeung ilustrasi.
- Seni visual Sunda Kuna boga hubungan jeung teks, kaweruh, kadatwan, profési, jeung kosmologi.

## Kudu Ditunda / Diwatesan

- Bentuk pasti seni lukis Pajajaran.
- Lokasi pasti balé tulis tanpa bukti tambahan.
- Hubungan langsung Falak Budha jeung Pajajaran.
- Bahan Falak Budha tanpa cek sumber primer.
- Perbandingan gaya final jeung tradisi séjén.
- Panel lukisan kadatwan tanpa sumber visual atawa arkeologis.

## Ditolak Lamun Teu Boga Sumber

- Pangucap lukis Pajajaran.
- Pangucap juru lukis Sunda Kuna.
- Warna sakral pasti tina Pajajaran.
- Simbol lukisan rahasia karuhun tanpa naskah atawa arsip.
