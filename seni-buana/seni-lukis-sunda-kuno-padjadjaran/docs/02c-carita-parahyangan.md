# Carita Parahyangan

Data pokok: Carita Parahyangan nyebut balé bobot pitu welas jajar, tinulis, pinarada, warnana cacaritaan.

Harti panalungtikan: data ieu penting pikeun maca hubungan balé, warna, jeung carita visual dina lingkungan kadatwan Sunda.

Tingkat sumber:

- A lamun kropak atawa faksimile langsung dicekel.
- C lamun maké artikel atawa kajian.

Watesan: harti visualna kudu diverifikasi deui.
