# Beda jeung Folder Tersaring

Folder `tersaring` eusina versi ringkes anu geus disaring.

Folder ieu eusina versi arsip anu leuwih lega.

## Tambahan dina paket ieu

- kode kapastian sumber
- ringkesan jumlah sumber
- pituduh umum Leiden / UBL / KITLV
- catetan beda angka jumlah naskah Sunda Leiden
- tambahan entri naskah Or. 3388
- tambahan entri naskah Or. 7722
- tambahan entri naskah Or. 8781
- tambahan entri naskah Or. 10.817
- tambahan audio Sunda pop
- tambahan koleksi gambar Asia Tenggara
- tambahan peta KITLV
- tambahan old prints, drawings, posters
- tambahan daftar kerja item Sunda di Digital Collections

## Putusan

Data ieu beda jeung folder tersaring, jadi disimpen dina folder sorangan.
