# Laku Adat jeung Ritual Valid

Dokumen ieu ngaganti file serpihan anu pondok teuing. Tujuanana pikeun ngumpulkeun laku adat jeung ritual dina hiji file anu bisa dibaca.

## Prinsip

Laku adat jeung ritual henteu otomatis jadi parentah prakték. Data bisa valid salaku catetan, tapi pamakéanana kudu nuturkeun sumber, tempat, komunitas, jeung watesan.

## Daftar Laku / Catetan

| Laku / Catetan | Status | Tingkat | Watesan |
|---|---|---|---|
| Ngaseuk | valid salaku istilah/laku adat lamun sumberna écés | C/D | kudu jelas komunitas jeung tempatna |
| Mipit | valid salaku istilah/laku adat lamun sumberna écés | C/D | ulah digeneralisasi |
| Seba Baduy | valid salaku adat hirup komunitas Baduy | C/D | kudu hormat kana wates komunitas |
| Kawalu | valid salaku adat hirup komunitas Baduy | C/D | bagian kawates teu meunang dibuka sembarangan |
| Seren Taun | valid salaku laku adat / upacara panén | C/D | variasi tempat kudu dibédakeun |
| Ngalaksa | valid salaku laku adat panén/pangan di sababaraha komunitas | C/D | cek sumber lokal |
| Hajat Lembur | valid salaku laku sosial-adat | C/D | kudu jelas lokasi jeung sumber |

## Aturan Pamakéan

1. Sebutkeun sumber jeung komunitas.
2. Ulah ngajarkeun ritual lamun teu aya idin jeung konteks.
3. Ulah nyebut hiji adat minangka adat Sunda umum lamun asalna ti komunitas husus.
4. Bagian sakral atawa kawates kudu dijaga.
5. Lamun ngan catetan kerja, paké tingkat E/F.

## Putusan

Laku adat anu geus kacatet henteu disebut palsu. Anu kudu diwatesan nyaéta pamakéan praktis jeung klaim umumna.
