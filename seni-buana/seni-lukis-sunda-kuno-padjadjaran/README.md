# Seni Lukis Sunda Kuna Jaman Pajajaran

Folder ieu keur paket panalungtikan ngeunaan **10 jenis / peta data seni lukis Sunda Kuna Jaman Pajajaran**.

Ieu lain dokumen nomer 10 dina runtuyan `Maca Tina Buana`. Ieu paket mandiri dina **Seni Buana** sabab eusina ngabahas seni visual, tulis, lukis, juru lukis, balé tulis, hiasan kadatwan, carita visual, gambar kosmologis, sumber naskah, jeung watesan panalungtikan.
