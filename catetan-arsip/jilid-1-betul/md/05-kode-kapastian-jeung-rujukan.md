# Kode Kapastian jeung Rujukan

Dokumen ieu ngajelaskeun kode kapastian pikeun maca Jilid 1 Betul.

## Kode Kapastian

| Kode | Harti | Pamakéan |
|---|---|---|
| A | naskah langsung, faksimile, transliterasi utama | Pustaka Jero kuat |
| B | katalog, metadata resmi, arsip, museum | ngabuktikeun ayana sumber |
| C | buku, jurnal, edisi, tarjamahan, kajian | rujukan tafsir jeung panalungtikan |
| D | catetan adat hirup anu kacatet | dipaké kalayan hormat jeung wates komunitas |
| E | rekonstruksi kiwari / tafsir anyar | henteu meunang disebut sumber kuna |
| F | catetan pribadi / kompilasi pamaké | bibit panalungtikan |
| X | klaim tanpa sumber / teu bisa diverifikasi | ditunda atawa ditolak |

## Aturan Nu Dibenerkeun

1. Naskah valid henteu otomatis X.
2. Naskah valid henteu otomatis ditunda.
3. Anu bisa ditunda nyaéta klaim rinci lamun sumberna can dicekel.
4. Pustaka Jero bisa ngandung sumber A/C lamun file atawa rujukan utamana geus écés.
5. Pustaka Luar dipaké pikeun sumber anu aya tapi can diasupkeun pinuh.

## Conto Bener

- Sanghyang Siksa Kandang Karesian: valid, A/C, Pustaka Jero.
- Sanghyang Hayu: valid, A/C, Pustaka Jero.
- Amanat Galunggung: valid, A/C, Pustaka Jero.
- Sewaka Darma: valid, A/C, Pustaka Jero.
- Carita Parahyangan: valid, A/C, Pustaka Jero.

## Conto Salah

- Naskah valid ditulis X tanpa alesan.
- Naskah valid ditulis ditunda padahal sumberna jelas aya.
- File dibagi jadi `bagian-*` pondok tanpa konteks.
- Klaim tafsir E disebut sumber kuna.

## Putusan

Pikeun folder ieu, file konsolidasi kudu jadi jalur baca utama. File serpihan disimpen ngan salaku jejak legacy, lain struktur utama.
