# Falak Budha / Raffles Java 1

Data pokok: Falak Budha dina Raffles Java 1 ngandung data rasi, tanda waktu, tabel, jeung unsur ilustrasi.

Harti panalungtikan: ieu penting pikeun maca gambar naskah kosmologis. Ieu leuwih deukeut kana gambar naskah tibatan lukisan dinding atawa panel kadatwan.

Tingkat sumber:

- B lamun maké katalog RAS.
- C lamun maké edisi Nurwansah jeung Griffiths 2025.
- A/B lamun gambar digital dianalisis langsung.

Watesan: hubungan langsung jeung Pajajaran kudu diwatesan nepi ka sumber primer dicekel.
