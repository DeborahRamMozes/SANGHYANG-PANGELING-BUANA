# catetan-arsip

Folder ieu pikeun nyimpen catetan arsip ngeunaan istilah, ritual, naskah, gambar, jeung sumber anyar.

## Format Catetan

```text
Judul:
Jenis Sumber:
Asal:
Pangarang / Penyusun:
Taun:
Tempat Arsip:
Tumbu / Nomor Katalog:
Basa / Aksara:
Kecap Konci:
Eusi Pokok:
Patali jeung Buana:
Patali jeung Karesian:
Patali jeung Kala / Wanci:
Patali jeung Kabuyutan:
Patali jeung Karuhun:
Status Verifikasi:
Catetan:
Rekomendasi:
```

## Rekomendasi
- diasupkeun ka `pustaka-jero/`
- disimpen ka `pustaka-luar/`
- ditunda
- ditolak
