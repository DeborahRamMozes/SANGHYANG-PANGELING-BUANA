# 08. Laku Sapopoé: Sabda, Bayu, Hedap, Eling

## Fungsi Dokumen

Dokumen ieu pikeun nyusun laku sapopoé ngaliwatan raga, sabda, bayu, hedap, rasa, eling, jeung lemah cai.

## Prinsip Laku

Laku sapopoé lain slogan spiritual anu dipajang di témbok terus poho pas nyanghareupan kahirupan. Laku kudu bisa dipigawé ku raga, kadéngé dina sabda, karasa dina bayu, katarima ku hedap, dijaga ku rasa, jeung dipulangkeun kana eling.

## Unsur Laku

| Unsur | Bacaan |
|---|---|
| Raga | Kaayaan awak, tanaga, nyeri, kabutuhan istirahat. |
| Sabda | Kecap anu diucapkeun, dijaga, atawa dipendem. |
| Bayu | Napas, gerak, tanaga, irama poé. |
| Hedap | Arah kahayang, niat, panggerak batin. |
| Rasa | Rasa anu muncul, disimpen, atawa kudu diberesihan. |
| Eling | Kasadaran kana diri, buana, jeung akibat laku. |
| Lemah Cai | Hubungan jeung tempat hirup, cai, taneuh, hawa, pangan. |

## Format Laku Harian

```text
Tanggal:
Wanci:
Kaayaan raga:
Sabda poé ieu:
Bayu / napas / gerak:
Hedap / niat:
Rasa:
Eling:
Lemah cai:
Laku anu dipigawé:
Tingkat sumber:
Catetan:
```

## Tingkat Sumber

- A: naskah/arsip primer.
- B: kajian ilmiah/budaya.
- C: tradisi lisan atawa pangalaman komunitas.
- D: tafsir budaya.
- E: hipotesis laku.
- F: catetan harian/puitik.
- X: klaim teu aya dasar.

## Catetan

Laku téh kudu turun kana poé. Lamun ngan luhur di kecap, éta lain laku. Éta hiasan kalimat anu make baju adat.
