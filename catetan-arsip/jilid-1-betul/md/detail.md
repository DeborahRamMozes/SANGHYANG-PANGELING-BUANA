# Detail Istilah Utama

## 1. AHUNG

Kategori: istilah / sora panggero dina bahan Sunda kuna.

Sumber asal:

- edisi `Tutur Bwana dan Empat Mantra Sunda Kuna`;
- koleksi Perpusnas RI L 620 / Peti 86 pikeun Tutur Bwana;
- edisi 2010 ku Tien Wartini, Ruhaliah, Mamat Ruhimat, Rahmat Sopian, jeung Aditia Gunawan.

Status:

- wangun `AHUNG` diverifikasi aya dina edisi naskah;
- harti rinci kudu nuturkeun teks jeung konteks sumber;
- henteu otomatis sarua jeung bentuk luar Sunda.

Tingkat kapastian: A/C.

Catetan:

A = wangun kecap aya dina edisi naskah Sunda kuna. C = tafsir harti dumasar bacaan/edisi jeung konteks.

## 2. Rambu pamakéan

Lamun teu aya sumber Sunda anu bisa dipariksa, hiji wangun ulah diasupkeun salaku kaweruh pasti.

Tingkat kapastian: E.

## 3. Pamisah istilah

Bentuk luar Sunda disimpen salaku catetan pembeda. Eta henteu dipaké salaku dasar otomatis pikeun ngajelaskeun `AHUNG`.

Tingkat kapastian: X pikeun panyaruakeun tanpa sumber langsung.
