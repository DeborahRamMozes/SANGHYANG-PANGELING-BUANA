# pangucap-suci

Folder ieu ngan pikeun pangucap anu boga sumber jelas. Pangucap anyar kudu dipisahkeun jeung dilabélan écés minangka pangucap anyar.

## Format Pangucap

```text
Judul pangucap:
Sumber:
Naskah / arsip:
Basa:
Aksara:
Transliterasi:
Tarjamahan Sunda kiwari:
Harti dina Sunda Buhun:
Pamakéan:
Watesan:
Tingkat kapastian:
```

## Aturan
Repo ieu henteu meunang nyieun doa, mantra, kidung, liturgi, atawa pangucap karuhun palsu.
