# kala-wanci

Folder ieu pikeun nyimpen catetan ngeunaan Kala Sunda, Wanci, Usum, Saka Sunda / Caka Sunda, Parocaang, Paropoek, cai, pare, gunung, jeung leuweung.

## Eusina
- catetan kala
- catetan wanci
- catetan usum
- catetan pare
- catetan cai
- catetan gunung
- catetan leuweung
- hubungan waktu jeung laku sapopoé

## Format Catetan

```text
Judul:
Kategori:
Tanggal / Wanci:
Sumber:
Tingkat kapastian:
Eusi Pokok:
Laku anu patali:
Catetan:
```
