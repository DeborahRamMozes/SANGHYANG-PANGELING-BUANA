# glosarium

Folder ieu pikeun nyimpen istilah Sunda Buhun jeung hartina dina pancer Buana, Wanci, Laku, Kabuyutan, jeung Karuhun.

## Format Entri

```text
Istilah:
Basa / Aksara:
Harti Sunda kiwari:
Harti dina Sunda Buhun:
Sumber:
Tingkat kapastian:
Catetan:
```

## Pangeling
Ulah nyampurkeun istilah anu geus aya sumberna jeung tafsir anyar tanpa tanda kapastian.
