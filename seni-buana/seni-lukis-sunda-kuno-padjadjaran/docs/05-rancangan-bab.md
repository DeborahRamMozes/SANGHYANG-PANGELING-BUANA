# Rancangan Bab Buku

1. Pancer seni rupa Sunda Kuna.
2. SSKK jeung ahli rupa.
3. Bale Tulis dina Carita Ratu Pakuan.
4. Bale Bobot dina Carita Parahyangan.
5. Falak Budha jeung Rupa Kala.
6. Motif kembang, alas, jeung tarate.
7. Jejak gambar pasca-Pajajaran.
8. Rekonstruksi anu jujur.
9. Rupa, Lemah Cai, jeung Karuhun.
10. Lampiran data GitHub.

## Catetan Arsip Ringkes

- SSKK: data tulis jeung ahli rupa.
- Carita Ratu Pakuan: data bale tulis.
- Carita Parahyangan: data bale, warna, jeung carita.
- Falak Budha / Raffles Java 1: data gambar kosmologis.
